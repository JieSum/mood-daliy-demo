# 心情日记 - 管理端Demo页面

## 页面概述

这是一个分析师视角的管理端页面，用于监测和管理多个用户的情绪状态。分析师可以查看用户的情绪记录、接收预警信息、发送提醒消息，并管理多个用户的情绪监测。

## 功能特点

- **多用户管理**：一个分析师可以管理多个人
- **预警监测**：实时显示用户的情绪预警信息
- **打卡提醒**：监测用户的打卡情况，及时提醒
- **消息发送**：直接向用户发送提醒和关怀消息
- **情绪分析**：查看用户的情绪趋势和统计数据

  "userId": "1",
  "message": "你最近的情绪状态不太好，记得照顾好自己，有什么需要可以随时联系我。",
  "type": "care"

  "messageId": "msg123",
  "sentAt": "2026-03-28T14:30:00Z",
  "userId": "1",
  "userName": "张三"

```

  "alertId": "alert1",

}
```

## 技术实现

### 前端
- React 18 + TypeScript
- Ant Design 组件库
- ECharts 图表库
- Tailwind CSS

### 后端
- Node.js + Express
- SQLite 数据库
- 实时消息推送

### 数据模型

**用户表**
- id: 用户ID
- name: 用户名
- status: 状态 (normal, high_risk)
- last_check_in: 最后打卡时间
- created_at: 创建时间

**预警表**
- id: 预警ID
- user_id: 用户ID
- type: 预警类型 (mild, moderate, severe)
- reason: 预警原因
- data: 预警数据
- is_read: 是否已读
- is_resolved: 是否已解决
- created_at: 创建时间
- resolved_at: 解决时间

**消息表**
- id: 消息ID
- user_id: 用户ID
- admin_id: 管理员ID
- content: 消息内容
- type: 消息类型 (reminder, care, alert)
- is_read: 是否已读
- created_at: 发送时间
