# 心情日记 - 单用户Demo页面

## 页面概述

这是一个极简的单用户模式心情日记页面，用户可以通过文字、语音或照片记录情绪，AI自动分析情绪状态，并查看历史记录和健康建议。

## 功能特点

- **极简界面**：只查看记录和健康建议
- **多模态输入**：支持文字、语音、照片
- **AI自动分析**：无需手动选择情绪
- **智能预警**：长期负面情绪自动提醒
- **历史记录**：按时间查看历史情绪

## 测试接口

### 1. 情绪记录接口

**POST** `/api/records`

**请求参数**:
```json
{
  "inputType": "text",
  "content": "今天感觉很开心，完成了一个重要的项目",
  "voicePath": null,
  "imagePaths": null
}
```

**响应**:
```json
{
  "id": "12345",
  "inputType": "text",
  "content": "今天感觉很开心，完成了一个重要的项目",
  "analysis": {
    "sentiment": "positive",
    "sentimentScore": 0.85,
    "intensity": 7,
    "keywords": ["开心", "完成", "项目"],
    "emotionTags": ["成就感", "满足"],
    "context": "工作相关的积极情绪",
    "confidence": 0.92
  },
  "createdAt": "2026-03-28T14:30:00Z"
}
```

### 2. 获取历史记录

**GET** `/api/records`

**响应**:

```json
      "intensity": 7,

      "confidence": 0.92

    "createdAt": "2026-03-28T14:30:00Z"
```

### 3. 获取健康建议

**GET** `/api/health-advice`

**响应**:
```json
{
  "recentStats": {
    "positive": 60,
    "negative": 30,
    "neutral": 10,
    "averageIntensity": 6.2

  "suggestions": [
```

### 4. 预警提醒接口

**GET** `/api/alerts`

**响应**:
```json
[
  {
    "id": "alert1",
    "type": "mild",
    "reason": "连续3天负面情绪",
    "data": {"days": 3},
    "isRead": false,
    "createdAt": "2026-03-27T10:00:00Z"
  }

```

### 5. 打卡提醒接口

**POST** `/api/reminders/check-in`

**请求参数**:
```json
{
  "userId": "current-user"
}
```

**响应**:
```json
{
  "status": "success",
  "message": "打卡成功",
  "lastCheckIn": "2026-03-28T14:30:00Z"
}
```

### 6. 消息发送接口

**POST** `/api/messages/send`

**请求参数**:
```json
{
  "userId": "current-user",
  "message": "记得今天记录你的情绪哦！",
  "type": "reminder"
}
```

**响应**:
```json
{
  "status": "success",
  "messageId": "msg123",
  "sentAt": "2026-03-28T14:30:00Z"
}
```

## 技术实现

### 前端
- React 18 + TypeScript
- Tailwind CSS + styled-components
- 响应式设计

### 后端
- Node.js + Express
- SQLite 数据库
- AI情绪分析服务

### AI分析
- 文本情绪分析：基于简化BERT模型
- 语音情绪分析：语音转文字 + 情绪识别
- 图像情绪分析：面部表情识别 + 场景分析
